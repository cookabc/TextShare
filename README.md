# 文字分享图生成器

一个简单的macOS应用，可以快速将剪贴板中的文字转换为美观的分享图片。

## 功能特点

- 🚀 **快速生成**: 按 ⌘⇧C 即可将剪贴板文字转换为图片
- 🎨 **三种主题**: 浅色、深色、渐变三种风格可选
- 📋 **自动复制**: 生成的图片自动复制到剪贴板
- 👀 **实时预览**: 提供预览窗口，可切换不同主题
- 💾 **一键保存**: 支持将图片保存为PNG格式

## 使用方法

1. **安装和运行**:
   ```bash
   cd TextToShare
   ./build.sh
   ./build/TextToShare
   ```

2. **生成分享图**:
   - 复制任意文本 (⌘C)
   - 按 ⌘⇧C 快捷键
   - 图片自动生成并复制到剪贴板
   - 点击菜单栏的📝图标可以打开预览窗口

3. **切换主题**:
   - 在预览窗口中可以选择不同主题
   - 支持浅色、深色、渐变三种风格
